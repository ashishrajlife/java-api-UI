About the Project

This is a sample project where I've implemented a Java API that performs CRUD (Create, Read, Update, Delete) operations. The API is fully functional and has been thoroughly tested using Postman to ensure its proper operation.

Below, you will find screenshots demonstrating how the API works in Postman. The images show various requests for performing CRUD operations on resources. Each image represents an individual API action, such as creating a resource, reading it, updating it, and deleting it.

Note: All actions have been successfully validated and tested, ensuring seamless integration and smooth performance.
